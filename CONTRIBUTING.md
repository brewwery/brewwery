# Contributing

Please keep changes scoped, readable, and macOS-first.

## Local development

1. Install pnpm.
2. Run `pnpm install`.
3. Run `pnpm dev`.

## Guidelines

- Prefer read-only Homebrew operations unless a feature explicitly needs mutation.
- Do not add telemetry, authentication, cloud sync, or monetization.
- Keep UI changes consistent with the dark macOS utility direction.
