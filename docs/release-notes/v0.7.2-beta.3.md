# Brewwery v0.7.2-beta.3

Private beta QA fixes for Search & Discovery.

This release focuses on package discovery correctness and responsiveness. Installed packages found through Homebrew search now show the correct installed state.

## What Changed

- Updated app version and metadata to `0.7.2-beta.3`.
- Kept `APP_CHANNEL` as `Private Beta`.
- Search results now compare formulae by `formula.name` and casks by `cask.token`.
- Search installed-state matching is case-insensitive.
- Package detail drawer respects the corrected installed state.
- Invalid search queries no longer run `brew search`.
- Cyrillic and non-allowlisted queries show a friendly local message.
- Stale search responses are ignored so old Homebrew output cannot overwrite newer queries.
- Search input remains responsive while Homebrew search runs.
- Rust search validation now uses the same allowlisted ASCII characters as the renderer.
- Updates page includes an explicit `Check for updates` action for `brew update`.
- The action requires confirmation, logs to History, and reloads outdated package counts.

## Validation Focus

```bash
pnpm typecheck
pnpm lint
pnpm --filter @brewwery/desktop build
pnpm package:mac:dir
```

Manual checks:

- Search `redis` when Redis is installed.
- Confirm Search status shows `Installed`.
- Open Redis details and confirm the primary action is `Uninstall`.
- Type Cyrillic into Search and confirm `brew search` is not triggered.
- Type quickly across multiple queries and confirm stale results do not replace newer ones.
- On Updates, run `Check for updates` and confirm the outdated count matches terminal `brew outdated`.

## Known Issues

- Builds are unsigned and not notarized.
- Local DMG creation can fail on this machine at `hdiutil create ... -fs APFS`.
- DMG should be produced by GitHub Actions on a clean macOS runner.
- Intel and universal builds are configured but still need dedicated validation.
- Cleanup and service actions show final operation output rather than live streaming progress.
- Package path and package-detail Open Terminal shortcuts are placeholders.
