# Brewwery v0.6.0-alpha.1

Private installable alpha for macOS.

Brewwery is a visual Homebrew manager for macOS. This alpha focuses on packaging reliability, Homebrew detection, package discovery, safe package actions, maintenance workflows, and private alpha QA coverage.

## Highlights

- Apple Silicon packaged `.app` build.
- Apple Silicon DMG and ZIP release artifact preparation.
- Homebrew detection for `/opt/homebrew/bin/brew`, `/usr/local/bin/brew`, and PATH fallback.
- Custom Homebrew path validation in Settings.
- Dashboard with progressive loading, service preview, update preview, and lightweight charts.
- Installed formulae and casks pages.
- Homebrew discovery search for formulae and casks.
- Package/cask detail drawer with metadata, dependencies, caveats, homepage, raw JSON, and safe actions.
- Install, uninstall, and upgrade actions with confirmation.
- Progress output for install, uninstall, and upgrade.
- Services, Cleanup, Doctor, Brewfile, History, and Settings workflows.
- Tray menu and keyboard shortcuts.
- First-launch onboarding.

## Install

1. Download `Brewwery-0.6.0-alpha.1-mac-arm64.dmg`.
2. Open the DMG.
3. Drag `Brewwery.app` to `/Applications`.
4. Open Brewwery from `/Applications`.
5. Accept the unsigned app warning if you trust this private alpha build.

## Uninstall

Quit Brewwery, then remove:

```bash
rm -rf "/Applications/Brewwery.app"
rm -rf "$HOME/Library/Application Support/Brewwery"
rm -f "$HOME/Library/Preferences/com.brewwery.app.plist"
rm -rf "$HOME/Library/Saved Application State/com.brewwery.app.savedState"
```

This does not remove Homebrew or any installed Homebrew packages.

## Known Issues

- The app is unsigned and not notarized.
- Apple Silicon is the primary tested architecture.
- Intel/universal builds may be untested.
- Some long-running Homebrew commands may appear quiet until Homebrew emits stdout/stderr.
- Cleanup and service actions do not stream live progress yet.
- Tapped formula names containing `/` are rejected by the current strict package-name validator.
- Package path and Open Terminal shortcuts from package detail are placeholders.
- No telemetry, crash reporting, cloud sync, or paid/pro features are included by design.

## Security Notes

- No generic shell command execution.
- No direct renderer `ipcRenderer` usage outside preload.
- Mutating actions require explicit confirmation.
- Rust command execution remains allowlisted.
- Streaming runner uses fixed argv arrays and `shell: false`.
- Custom Homebrew path must pass validation before use.
- No `sudo`.
- No automatic cleanup, install, uninstall, or upgrade.
