# Brewwery v0.9.1

Brewwery v0.9.1 adds Discover and Favorites as Release Candidate polish.

## Highlights

- Discover curated Homebrew formulae and casks.
- Save favorite formulae and casks locally.
- Open package details from Discover and Favorites.
- Install available Discover packages through the existing confirmation flow.
- See favorite indicators in Search, Packages, Casks, Discover, and package details.

## Security Notes

- Discover is a static local registry.
- Favorites are stored locally.
- No new brew commands were added.
- No new generic command runner was added.
- No telemetry, cloud sync, account system, paid logic, or donation/support logic was added.

## Known Limitations

- Builds remain unsigned and not notarized during Release Candidate testing.
- Local DMG creation can still fail on some machines because of the documented `hdiutil` APFS issue.
- Intel and universal builds remain untested unless a proper x64 Rust target and Intel validation environment are available.
